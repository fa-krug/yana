# App Store listing copy

Marketing descriptions for the App Store, one file per locale. Each is plain copy
ready to paste into App Store Connect and stays within Apple's 4000-character limit.

- [description-en.md](description-en.md) — English (en)
- [description-de.md](description-de.md) — German (de)

When adding a language, mirror the structure of the English file and keep every
locale under 4000 characters.
