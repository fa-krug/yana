# App Store listing copy

Marketing descriptions for the App Store, one file per locale. Each is plain copy
ready to paste into App Store Connect and stays within Apple's 4000-character limit.

Descriptions (max 4000 characters each):

- [description-en.md](description-en.md) — English (en)
- [description-de.md](description-de.md) — German (de)

Keywords for the App Store Connect keyword field. One comma-separated line, no
spaces, max 100 characters each:

- [keywords-en.txt](keywords-en.txt) — English (en)
- [keywords-de.txt](keywords-de.txt) — German (de)

When adding a language, mirror the structure of the English files and keep each
description under 4000 characters and each keyword line under 100.
